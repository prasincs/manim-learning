# Educational Resources & References

Comprehensive list of MOOCs, courses, and educational content to reference and link in your videos.

## Zero-Knowledge Proofs

### ZK-Learning (zklearning.org)
- **URL**: https://zk-learning.org/
- **Description**: Comprehensive course on zero-knowledge proofs
- **Content**: Video lectures, slides, and exercises
- **Topics**: zk-SNARKs, zk-STARKs, Bulletproofs, PLONK
- **Best for**: Phase 5 modules (Advanced Cryptography)

### ZK Hack Whiteboard Sessions
- **URL**: https://zkhack.dev/whiteboard/
- **Description**: Whiteboard-style explanations of ZK concepts
- **Content**: Short video explainers by experts
- **Topics**: Polynomial commitments, Fiat-Shamir, IOP
- **Best for**: Phase 5 modules, visual reference for explanations

### ZK Podcast
- **URL**: https://www.zeroknowledge.fm/
- **Description**: Podcast covering ZK and blockchain topics
- **Best for**: Keeping up with latest developments

### Ingopedia (Ingo's ZK Explainers)
- **URL**: https://twitter.com/ingonyama (and blog posts)
- **Description**: Clear visual explanations of ZK concepts
- **Best for**: Inspiration for visual styles

When using these resources in your videos:
